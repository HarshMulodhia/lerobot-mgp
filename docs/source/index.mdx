# LeRobot-MGP

**MGP-first real-world robotics on top of LeRobot**

🤗 LeRobot-MGP extends LeRobot with a first-class Markov Generator Policy (`mgp`) workflow for real-world data collection, training, and deployment.

🤗 This fork keeps upstream LeRobot compatibility while focusing documentation and usage around MGP.

🤗 You can still use the original LeRobot policy ecosystem, datasets, and environments.

🤗 LeRobot hosts pretrained models and datasets on the LeRobot HuggingFace page.
